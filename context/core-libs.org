[FIX id:fix.core-libs.enum-instance-ownership-2026-07-17 date:2026-07-17 by:codex src:core/prelude/Data/Enum.mon,codegen.c,main.c conf:high]
  =succ=, =pred=, =toEnum=, =fromEnum=, and finite enumeration are owned by
  the =Enum= typeclass.  =Int= is now an ordinary =Enum Int= instance rather
  than a set of monomorphic definitions shadowing those methods, so users can
  define and call =Enum= methods for their own types.  The eager core exposes
  only finite enumeration methods; Haskell's infinite =enumFrom= operations
  require a lazy stream abstraction and must not masquerade as strict lists.
  The dependent shadow pass defers typeclass calls to HM/typeclass checking so
  =Enum.succ= is not mistaken for the dependent =Nat= constructor.

[FIX id:fix.core-libs.enum-minimal-defaults-2026-07-17 date:2026-07-17 by:codex src:core/prelude/Data/Enum.mon,typeclass.c,codegen.c,env.c,runtime.c,runtime.h conf:high]
  The minimal complete =Enum= definition is now =toEnum= plus =fromEnum=.
  =succ=, =pred=, contiguous finite ranges, and stepped finite ranges are lawful
  class defaults specialized to each instance.  Result-directed calls such as
  =toEnum :: Int -> a= prefer the enclosing instance, recursive defaults retain
  their concrete ADT ABI, zero-step finite ranges fail explicitly, and range
  termination avoids converting values beyond the endpoint.  Runtime opaque
  boxing lets user ADT values inhabit generic core collections safely.

[DEC id:dec.core-libs.canonical-typeclass-ownership-2026-07-18 date:2026-07-18 by:codex conf:high]
  Generic operations are canonically owned by their typeclasses, following the
  Haskell model. =Numeric.Num= owns arithmetic such as =abs/signum/inc/square=;
  =Numeric.Integral= owns parity, division, =gcd=, and =lcm=; =Data.Ord.Ord=
  owns comparison, =min/max/clamp=, and interval predicates. Primitive modules
  such as =Data.Int= and =Data.Float= contain only genuinely type-specific
  vocabulary and must not duplicate those implementations. =Math= owns
  algorithms and constants, not aliases of typeclass methods.
