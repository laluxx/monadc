[DEC id:dec.debug.diagnostics-should-be-specific date:2026-06-16 by:llm]
  Diagnostic tests are useful only when they assert stable error classes and
  source spans. Until diagnostic text stabilizes, quarantine text-sensitive cases
  as =known-fail= rather than deleting them.
