This page covers boolean logic in Monad: =and=, =or=, =not=, truthiness
rules, and type predicates.

Boolean logic operators combine truth values. =and= and =or= use
short-circuit evaluation — the second argument is only evaluated if
needed.

See [[file:booleans.org][Booleans]] for the Bool type and [[file:conditionals.org][Conditionals]] for control flow
using boolean conditions.

* 1. and

Logical conjunction — returns =True= if all arguments are true:

#+BEGIN_SRC monad
(and True True)                  ;; True
(and True False)                 ;; False
(and (> x 0) (< x 100))         ;; True if x in range
#+END_SRC

#+BEGIN_SRC wisp
and True True
and True False
and
  > x 0
  < x 100
#+END_SRC

Short-circuit: if any argument is =False=, remaining arguments are
not evaluated. Variadic: 2..N arguments.

See also: [[file:conditionals.org][Conditionals]]

* 2. or

Logical disjunction — returns =True= if any argument is true:

#+BEGIN_SRC monad
(or False True)                  ;; True
(or False False)                ;; False
(or (< x 0) (> x 100))          ;; True if x out of range
#+END_SRC

#+BEGIN_SRC wisp
or False True
or False False
or
  < x 0
  > x 100
#+END_SRC

Short-circuit: if any argument is =True=, remaining arguments are not
evaluated. Variadic: 2..N arguments.

* 3. not

Logical negation:

#+BEGIN_SRC monad
(not True)                       ;; False
(not False)                      ;; True
(not (= x 0))                   ;; True if x != 0
#+END_SRC

#+BEGIN_SRC wisp
not True
not False
not (= x 0)
#+END_SRC

=not= takes a single argument.

See also: [[file:booleans.org][Booleans]]

* 4. Truthiness

Only =False= and =nil= are considered falsey in condition position. All
other values are truthy:

#+BEGIN_SRC monad
(if 0 "truthy" "falsey")        ;; "truthy" — 0 is truthy
(if "" "truthy" "falsey")       ;; "truthy" — "" is truthy
(if False "truthy" "falsey")    ;; "falsey"
(if nil "truthy" "falsey")      ;; "falsey"
#+END_SRC

#+BEGIN_SRC wisp
if 0 "truthy" "falsey"
if "" "truthy" "falsey"
if False "truthy" "falsey"
if nil "truthy" "falsey"
#+END_SRC

See [[file:conditionals.org][Conditionals]] for =if=, =cond=, =when=, =unless=.

Key Points:
- Only =False= and =nil= are falsey; all other values are truthy (including 0 and "")
- =and= and =or= use short-circuit evaluation = second argument evaluated only if needed
- =not= takes a single argument and returns Boolean

* 5. Operator Table

| Operator | Example        | Description           | Short-circuit |
|----------+----------------+-----------------------+---------------|
| and      | =(and a b)=    | Logical AND           | Yes           |
| or       | =(or a b)=     | Logical OR            | Yes           |
| not      | =(not a)=      | Logical NOT           | N/A           |

* 6. Type Predicate

#+BEGIN_SRC monad
(Bool? (and True False))   ;; True
(Bool? (not True))          ;; True
(Bool? 42)                  ;; False
#+END_SRC

#+BEGIN_SRC wisp
Bool? (and True False)   ;; True
Bool? (not True)          ;; True
Bool? 42                  ;; False
#+END_SRC

See [[file:booleans.org][Booleans — Type Predicates]].
