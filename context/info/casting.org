Test coverage: type conversions are exercised comprehensively in
=Main.mon= (lines 1103-1215) and by =rt_cast_*.mon= fixtures.

Type names are callable as conversion functions. =(TargetType value)=
converts =value= to the target type.

This page covers type casting in Monad: numeric conversions, fixed-width
conversions, Char/String conversions, and type predicates.

See [[file:numeric.org][Numeric Types]] for the type hierarchy and [[file:core.org][Core Library]] for the
built-in function dispatch mechanism.

* 1. Numeric Conversions

Conversion between numeric types is explicit. No implicit coercions
beyond the Int/Float mixed-operation promotion.

#+BEGIN_SRC monad
(Int 3.14)                       ;; 3 — truncation
(Float 42)                      ;; 42.0
(Hex 255)                       ;; "FF"
(Oct 64)                        ;; "100"
(Bin 7)                        ;; "111"
#+END_SRC

#+BEGIN_SRC wisp
Int 3.14
Float 42
Hex 255
Oct 64
Bin 7
#+END_SRC

See [[file:arithmetic.org][Arithmetic — Mixed Int/Float]] for promotion rules.

Key Points:
- All numeric conversions are explicit = no implicit coercions beyond Int/Float mixed ops
- =(Int 3.14)= truncates toward zero; =(Float 42)= promotes to 64-bit float
- =(Hex 255)=, =(Oct 64)=, =(Bin 7)= format as string representations
- Type names double as conversion functions: =(TargetType value)=

See also: [[file:arithmetic.org][Arithmetic]]

* 2. Fixed-Width Conversions

Fixed-width types have the same size as their LLVM counterparts and
wrap on overflow (two's complement). See [[file:types.org][Types]] and [[file:ffi.org][C FFI]].

#+BEGIN_SRC monad
(I8 128)                         ;; -128 (wraps)
(U8 255)                        ;; 255
(I16 1000)                      ;; 1000
(U16 65535)                     ;; 65535
(I32 100000)                    ;; 100000
(U32 4000000000)                ;; 4000000000
(I64 -1)                        ;; -1
(U64 18446744073709551615)      ;; max U64
(I128 x)                        ;; 128-bit signed
(U128 x)                        ;; 128-bit unsigned
(F32 3.14159265358979)          ;; float (32-bit)
#+END_SRC

#+BEGIN_SRC wisp
I8 128
U8 255
I16 1000
U16 65535
I32 100000
U32 4000000000
I64 -1
U64 18446744073709551615
I128 x
U128 x
F32 3.14159265358979
#+END_SRC

All fixed-width conversions dispatch through strcmp handlers
in [[file:../context/codegen-ops.org][context/codegen-ops.org]].

* 3. Char Conversions

=Char= converts between codepoint integer and character. See [[file:characters.org][Characters]].

#+BEGIN_SRC monad
(Char 65)                        ;; 'A'
(Char 0x3C0)                    ;; 'π'
(Int 'A')                       ;; 65
#+END_SRC

#+BEGIN_SRC wisp
Char 65
Char 0x3C0
Int 'A'
#+END_SRC

* 4. String Conversions

=String= produces the display representation of any value. See [[file:strings.org][Strings]].

#+BEGIN_SRC monad
(String 42)                      ;; "42"
(String 3.14)                   ;; "3.14"
(String 'A')                    ;; "A"
#+END_SRC

#+BEGIN_SRC wisp
String 42
String 3.14
String 'A'
#+END_SRC

* 5. Conversion Table

| Syntax          | Description                  |
|-----------------+------------------------------|
| =(Int x)=         | Convert to signed 63-bit int |
| =(Float x)=       | Convert to 64-bit float      |
| =(Char x)=        | Convert integer to character |
| =(String x)=      | Convert to string            |
| =(Hex x)=         | Format as hex string         |
| =(Oct x)=         | Format as octal string       |
| =(Bin x)=         | Format as binary string      |
| =(I8/U8 x)=       | 8-bit conversion             |
| =(I16/U16 x)=     | 16-bit conversion            |
| =(I32/U32 x)=     | 32-bit conversion            |
| =(I64/U64 x)=     | 64-bit conversion            |
| =(I128/U128 x)= | 128-bit conversion           |
| =(F32 x)=         | 32-bit float conversion      |

Key Points:
- Every type name is callable as a conversion function: =(TargetType value)=
- Fixed-width types wrap on overflow (two's complement)
- =Char= converts between codepoint integer and character
- =String= produces the display representation of any value
- =Hex=, =Oct=, =Bin= format as string representations of the numeric value

See also: [[file:ffi.org][C FFI]]

* 6. Type Predicates for Convertibility

#+BEGIN_SRC monad
(Int? (Int 3.14))     ;; True
(Char? (Char 65))     ;; True
(String? (String 42)) ;; True
#+END_SRC

#+BEGIN_SRC wisp
Int? (Int 3.14)     ;; True
Char? (Char 65)     ;; True
String? (String 42) ;; True
#+END_SRC
