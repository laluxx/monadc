[DOC id:doc.category.tui.screen-model src:context/category/tui.org conf:high]
  The screen has four regions: query header, left result pane, right focus pane,
  and footer.  The left pane is a ranked live search result set.  The right pane
  is an object card with metadata and Hom-set arrows.  =Tab= toggles focus; the
  active pane is brighter.
