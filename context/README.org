[DEC id:dec.readme.read-context-first date:2026-06-16 by:human+llm]
  Before changing compiler behavior or tests, read the relevant topic file(s),
  the linked context nodes, and the test metadata. After changing behavior,
  update the context with either a new decision, a superseding note, or a
  quality warning. No silent garden drift.

[DEC id:dec.readme.evidence-before-confidence date:2026-06-16 by:llm]
  High-confidence records require source code, test output, compiler output, or
  an explicit user decision. Inferences must be marked =INF= and linked to their
  observations. Opinions belong in =THINK= or =IDEA=, not =OBS=.
