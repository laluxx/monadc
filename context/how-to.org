[OBS id:obs.how-to.file src:how_to/ conf:high]
  The =how_to/= directory contains example programs demonstrating Monad
  features: ADTs, iterators, macros, syntax variants, refinement types,
  memory layouts, external library bindings (GLFW, Raylib, SDL), Vulkan
  terminal rendering, and scratch experiments. These serve as both
  documentation and integration test cases for the compiler.

  pattern matching, data declarations, layout structs, module imports,
  inline C includes, type annotations.
