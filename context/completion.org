#+CAPTION: Tab Completion & Usage Category Contract
| Field | Contract |
|-------+----------|
| Owned phase | CLI help text generation and tab-completion infrastructure — print_usage, print_subcommand_menu, and future shell-completion generators. |
| Inputs | CLI subcommand name, program name from argv[0]. |
| Outputs | Formatted help text printed to stdout. |
| Invariants | Output must remain in sync with cli.h subcommand enum; never crash on invalid subcommand input. |
| Test hooks | CLI integration tests, help text golden-file comparisons. |
| Failure symptoms | Wrong subcommand listed, missing option, garbled help text, crash on --help. |
| Context neighbors | cli, main |

[DEC id:dec.completion.category-contract-owner date:2026-06-18 by:big-pickle]
  Failures matching the symptoms above should start triage in this category, then
  follow the listed neighbors only when the local contract does not explain the
  issue.
