[OBS id:obs.buildsystem.highlevel.project src:buildsystem.c:398-422 conf:high]
  =build_project(source_path, output_name, verbose, force_rebuild)= → =bool= —
  Creates a =BuildContext=, sets verbose/force_rebuild, compiles the source with
  all dependencies via =build_compile_with_deps=, links all artifacts via
  =build_link_executable=, then frees the context. Returns true on success.

#+CAPTION: Build System Architecture Flow
| Step | Function                       | Description                           |
|------+--------------------------------+---------------------------------------|
| 1    | =build_project=                  | Entry point: creates BuildContext      |
| 2    | =build_compile_with_deps=        | Resolves source path, starts recursion |
| 3    | =compile_module_recursive=       | Extracts imports, recurses, compiles   |
| 4    | =build_link_executable=          | Links all .o files with runtime lib    |

  connects-to -> `monadc.context.cli.cmd-build` (=cmd_build= from cli.c invokes
