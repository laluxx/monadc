[DEC id:dec.language.stdout-golden date:2026-06-11 by:human]
Executable fixtures may have a sibling =.stdout= file. When present, the test
runner executes the compiled program and requires the program output to end with
that file exactly, allowing stable suffix checks while ignoring noisy compiler
output.
