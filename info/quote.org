Test coverage: quote and quasiquote are exercised by codegen fixtures
=rt_quote_*.mon= and =rt_quasiquote_*.mon=.

Quote and quasiquote prevent evaluation, treating the expression as
literal data. They are the foundation of code-as-data macros.

* quote

Prevents evaluation of the following expression:

#+BEGIN_SRC monad
(quote (1 2 3))                  ;; '(1 2 3) тАФ list literal
'(1 2 3)                         ;; same, shorthand syntax
'x                               ;; symbol x (not variable lookup)
'()                              ;; empty list
#+END_SRC

#+BEGIN_SRC wisp
quote (1 2 3)
'(1 2 3)
'x
'()
#+END_SRC

The shorthand ='expr= is reader sugar for =(quote expr)=. The reader
produces =AST_QQUOTE= nodes for quote (reader.h).

Use cases:
- Create list literals: ='(a b c)=
- Refer to symbols as data: ='my-symbol=
- Prevent macro expansion: ='(if x y z)=

#+BEGIN_SRC monad
(define my-list '(1 2 3))
(car my-list)                    ;; 1
(define s 'hello)                ;; symbol, not variable reference
#+END_SRC

#+BEGIN_SRC wisp
define my-list '(1 2 3)
car my-list
define s 'hello
#+END_SRC

* quasiquote

Template with unquote holes. Uses corner bracket syntax:

#+BEGIN_SRC monad
тМЬ1 2 3тМЭ                       ; quasiquote = (quasiquote 1 2 3)
тМЬ1 2 тМЮxтМЯ 3тМЭ                   ; unquote: тМЮexprтМЯ = (unquote expr)
#+END_SRC

#+BEGIN_SRC wisp
тМЬ1 2 3тМЭ
тМЬ1 2 тМЮxтМЯ 3тМЭ
#+END_SRC

=тМЬ= / =тМЭ= delimit the quasiquote (U+231C / U+231D).
=тМЮ= / =тМЯ= delimit unquote holes (U+231E / U+231F).

No backtick syntax тАФ use corner brackets.

* unquote-splicing

Splice a list into the surrounding template:

#+BEGIN_SRC monad
(define xs '(2 3))
тМЬ1 тМЮ,@xsтМЯ 4тМЭ                   ;; (1 2 3 4)
#+END_SRC

#+BEGIN_SRC wisp
define xs '(2 3)
тМЬ1 тМЮ,@xsтМЯ 4тМЭ
#+END_SRC

=тМЮ,@xsтМЯ = =(unquote-splicing xs)= тАФ flattens the list into the
surrounding template.

* Macro usage

Quasiquote is the primary macro output template:

#+BEGIN_SRC monad
(define (when condition . body -> Syntax)
  тМЬif condition (begin тМЮ,@bodyтМЯ)тМЭ)
#+END_SRC

#+BEGIN_SRC wisp
define (when condition . body -> Syntax)
  тМЬif condition (begin тМЮ,@bodyтМЯ)тМЭ
#+END_SRC

Inside the quasiquote, unquoted expressions (=тМЮexprтМЯ=) are evaluated
at macro-expansion time and spliced into the output.
