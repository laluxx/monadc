* Package layout

Convention (not enforced):

#+BEGIN_SRC
my-pkg/
  my-pkg.mon        # main entry, exports API
  my-pkg-extra.mon  # optional submodules
  tests/
    test-foo.mon
  README.md
#+END_SRC

* Module Path Resolution

See [[file:modules.org][Module System]] for path resolution rules.

* Versioning

No built-in package manager or versioning. Package management is
manual =git= + =monad build=.

* Publishing

No package registry exists. Distribute source via git or tarball.
Future: planned package manager.
