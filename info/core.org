Built-in functions with =strcmp= dispatch in codegen. Always available
without import. This page provides an overview; each category has a
dedicated reference page.

Test coverage: exercised by 423 codegen runtime fixtures in
[[file:../tests/codegen/][tests/codegen/]] and =Main.mon=
(1244 lines).

|------------+-------+----------------------------------|

| =dissoc=   | 2     | Remove key (immutable)           |
| =dissoc!=  | 2     | Remove key (in place)            |
| =find=     | 2     | Lookup key, returns (key val) or nil |

* Usage Examples

Arithmetic:

#+BEGIN_SRC monad
(+ 1 2 3)          ;; => 6
(* 2 5)            ;; => 10
(- 10 4 3)         ;; => 3
(/ 12 4)           ;; => 3
#+END_SRC

#+BEGIN_SRC wisp
+ 1 2 3            # => 6
* 2 5              # => 10
- 10 4 3           # => 3
/ 12 4             # => 3
#+END_SRC

Printing with =show=:

#+BEGIN_SRC monad
(show "hello, world")
(show "the sum is " (+ 1 2))
#+END_SRC

#+BEGIN_SRC wisp
show "hello, world"
show "the sum is " + 1 2
#+END_SRC

Assertions:

#+BEGIN_SRC monad
(assert-eq (+ 2 2) 4 "two plus two")
(assert-eq (* 3 3) 9 "three squared")
#+END_SRC

#+BEGIN_SRC wisp
assert-eq (+ 2 2) 4 "two plus two"
assert-eq (* 3 3) 9 "three squared"
#+END_SRC

Type conversion:

#+BEGIN_SRC monad
(Int 3.14)       ;; => 3
(Float 42)       ;; => 42.0
#+END_SRC

#+BEGIN_SRC wisp
Int 3.14         # => 3
Float 42         # => 42.0
#+END_SRC

Aborting with an error:

#+BEGIN_SRC monad
(error "something went wrong")
#+END_SRC

#+BEGIN_SRC wisp
error "something went wrong"
#+END_SRC

List operations:

#+BEGIN_SRC monad
(define lst (cons 1 (cons 2 nil)))
(car lst)            ;; => 1
(cdr lst)            ;; => (2)
(count lst)          ;; => 2
(head lst)           ;; => 1
(empty? nil)         ;; => #t
#+END_SRC

#+BEGIN_SRC wisp
define lst cons 1 (cons 2 nil)
car lst               # => 1
cdr lst               # => (2)
count lst             # => 2
head lst              # => 1
empty? nil            # => #t
#+END_SRC

Map operations:

#+BEGIN_SRC monad
(define m #{:a 1 :b 2})
(find m :a)       ;; => (:a 1)
(keys m)          ;; => (:a :b)
(vals m)          ;; => (1 2)
(count m)         ;; => 2
#+END_SRC

#+BEGIN_SRC wisp
define m #{:a 1 :b 2}
find m :a         # => (:a 1)
keys m            # => (:a :b)
vals m            # => (1 2)
count m           # => 2
#+END_SRC

String operations:

#+BEGIN_SRC monad
(count "hello")         ;; => 5
(starts-with? "hello" "he")   ;; => #t
(ends-with? "hello" "lo")     ;; => #t
(contains? "hello" "ll")      ;; => #t
#+END_SRC

#+BEGIN_SRC wisp
count "hello"               # => 5
starts-with? "hello" "he"   # => #t
ends-with? "hello" "lo"     # => #t
contains? "hello" "ll"      # => #t
#+END_SRC
