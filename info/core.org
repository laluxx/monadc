Built-in functions with =strcmp= dispatch in
[[file:../codegen.c]] =codegen_expr=. Always available without
import. Functions registered in =register_builtins= only (NOT
callable from paren-mode) are marked as such.

Dispatch architecture ([[file:../context/language.org::176]]):

1. =codegen_expr= switches on AST type
2. For =AST_LIST=, strcmp on head symbol (83+ symbols)
3. =register_builtins= registers 94 entries — only callable if also
   has a strcmp handler
