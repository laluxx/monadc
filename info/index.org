* Key Points

- Monad is a compiled Lisp-family language with dual syntax (S-expr + Wisp).
- The compiler is written in C (~29,000+ lines) and targets x86-64 Linux via LLVM 19+.
- Memory management uses arena allocation with bump-pointer semantics — no GC.
- The type system combines Hindley-Milner inference with dependent types (ITT/NbE).
- Macros are ordinary functions returning =Syntax= values — no separate =macro= keyword.
- All reference pages link to compiler-internal context docs for cross-referencing.
