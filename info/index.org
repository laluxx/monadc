This is the Monad language reference. It covers the language from the
user's perspective — syntax, types, standard library, tooling — NOT the
compiler internals (those live in [[file:../context.org][context/]]).

Written in Org mode. Export to GNU Info via Emacs: ~C-c C-e i i~.
Monad version: 0.1.0. Compiler source: [[file:../main.c][main.c]] (1432 lines).

* Conventions

Throughout this reference:

- =(code)= marks Lisp-style S-expression syntax
- ~indented~ blocks mark Wisp (whitespace-aware) syntax
- =(...)=, =[...]=, ={...}= are literal syntax
- =(italic text)= denotes metavariables
- =(type::value)= shows inferred types

Cross-references to the compiler context link to [[file:../context.org][context/]] files.
