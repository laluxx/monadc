** What does Monad code look like?

#+BEGIN_SRC monad
(define (factorial n -> Int)
  (if (<= n 1)
    1
    (* n (factorial (- n 1)))))

(show (factorial 10))  ;; prints 3628800
#+END_SRC

#+BEGIN_SRC wisp
define (factorial n -> Int)
  if (<= n 1)
    1
    * n (factorial (- n 1))

show (factorial 10)    # prints 3628800
#+END_SRC

** How do I call C functions?

#+BEGIN_SRC monad
(include "math.h")
(show (sqrt 144.0))   ;; prints 12.0
#+END_SRC

#+BEGIN_SRC wisp
include "math.h"
show (sqrt 144.0)     # prints 12.0
#+END_SRC

** Can I use two syntaxes in one file?

Yes — S-expressions and Wisp mix freely within a single file:

#+BEGIN_SRC monad
(define x 5)
# this is wisp
set! x 10
(show x)
#+END_SRC
