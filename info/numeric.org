Monad provides a comprehensive numeric type hierarchy spanning arbitrary-precision concepts down to fixed-width machine types for C interop.

* Primary Numeric Types

| Type | Description | Example | Precision |
|------+-------------+---------+-----------|
| =Int= | Architecture-width signed | =42=, =-7= | 64-bit (x86-64) |
| =Float= | IEEE 754 double | =3.14=, =1e10= | 64-bit |
| =Ratio= | Exact rational | =1/3=, =50%= | Arbitrary (num/den) |

See [[file:rationals.org][Rationals]] for the =Ratio= type.

* Fixed-Width Numeric Types

For C interop and bit-level control ([[file:types.org][Types]]):

| Type | Width | Description |
|------+-------+-------------|
| =I8= | 8 | Signed byte |
| =U8= | 8 | Unsigned byte |
| =I16= | 16 | Short |
| =U16= | 16 | Unsigned short |
| =I32= | 32 | Int |
| =U32= | 32 | Unsigned int |
| =I64= | 64 | Long |
| =U64= | 64 | Unsigned long |
| =I128= | 128 | Quad word |
| =U128= | 128 | Unsigned quad |
| =F32= | 32 | IEEE 754 single |
| =F80= | 80 | x86 extended precision |

Note: =Int= maps to the machine word (i64 on x86-64). Use fixed-width types for explicit layout control or FFI.

* Arithmetic Operations ([[file:core.org][Core Library]])

All working on Int, Float, and mixed operands:

| Operation | Example | Variadic |
|-----------+---------+----------|
| =+= | =(+ 1 2 3)= | Yes (1..N) |
| =-= | =(- 10 3)= | Yes (1..N) |
| =*= | =(* 2 3 4)= | Yes (1..N) |
| =/= | =(/ 10 3)= | Yes (1..N) |
| =%= | =(% 10 3)= | No (binary) |

* Bitwise Operations

| Operation | Example | Variadic |
|-----------+---------+----------|
| =&= | =(& 0xFF 0x0F)= | Yes |
| =bit-or=, =∨= | =(bit-or 1 2 4)= | Yes |
| =bit-xor=, =⊕= | =(bit-xor 5 3)= | Yes |
| =~= | =(~ 0xFF)= | Unary |
| =<<= | =(<< 1 4)= | Binary |
| =>>= | =(>> 16 2)= | Binary |
| =>>>= | =(>>> -1 1)= | Binary |

* Comparison Operations

| Operation | Example | Variadic |
|-----------+---------+----------|
| ==== | =(= 1 1 1)= | Yes (2..N) |
| =!= | =(!= 1 2)= | Yes |
| =<= | =(< 1 2 3)= | Yes |
| =<== | =(<= 1 2)= | Yes |
| =>= | =(> 3 2 1)= | Yes |
| =>== | =(>= 3 3)= | Yes |
