Control forms are expressions (they return values). Codegen dispatch
at [[file:../codegen.c::7471]].

|--------+-------------------------------------+-----------------------|
| if     | =(if cond t f)=                       | [[file:../codegen.c::7572]] |
| cond   | =(cond [t1 e1] ...)=                  | compiler macro        |
| unless | =(unless cond body ...)=              | builtin               |
| while  | =(while cond body ...)=               | [[file:../codegen.c::7957]] |
| begin  | =(begin e1 e2 ...)=                   | [[file:../codegen.c::7561]] |

** Mutation

| Form | Example                  | Codegen source          |
|------+--------------------------+-------------------------|
| set! | =(set! x 42)=            | [[file:../codegen.c::7471]] |
| <-   | =(<- x 42)=              | desugars to =set!=      |
| set! | =(set! obj.field val)=  | [[file:../codegen.c::7471]] |
| <-   | =(<- obj.field val)=    | desugars to =set!=      |

The =<-= sugar (=x <- 42=) desugars to =(set! x 42)=
([[file:../reader.c]]). Valid in both S-expression and Wisp syntax.

** Null-coalescing

=(?? expr default)= ([[file:../codegen.c::5763]]) returns =expr= if not
=nil=, =default= otherwise.
