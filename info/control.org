All control forms are expressions — they return values. Codegen dispatch for
control forms begins at [[file:../codegen.c::7471][codegen.c:7471]].

Test coverage: control flow is exercised by =rt_control_all.mon=,
=rt_control_comp.mon=, and 35+ diagnostic error fixtures.

This page provides an overview of Monad's control flow forms. Each
group has a dedicated reference page.

* Working Control Forms

These forms have strcmp dispatch handlers and are callable from paren-mode:

| Form   | Example                 | Codegen source                           |
|--------+-------------------------+------------------------------------------|
| if     | =(if cond t f)=           | [[file:../codegen.c::7572]]                  |
| unless | =(unless cond body ...)=  | [[file:../codegen.c::7919]]                  |
| while  | =(while cond body ...)=   | [[file:../codegen.c::7957]]                  |
| for    | =(for [i 0 10] (show i))= | [[file:../codegen.c::8004]]                  |
| begin  | =(begin e1 e2 ...)=       | [[file:../codegen.c::7561]]                  |
| do     | =(do expr ...)=           | alias for =begin=, [[file:../codegen.c::7830]] |
| when   | =(when test body ...)=    | [[file:../codegen.c::7879]]                  |
| with   | =(with [x 1 y 2] body)=   | [[file:../codegen.c::7841]]                  |

* Known Codegen Gaps

The following forms have no strcmp dispatch handler:

| Form   | Example                         | Status                                    |
|--------+---------------------------------+-------------------------------------------|
| cond   | =(cond [t1 e1] [t2 e2] [else e])= | No strcmp dispatch — not yet implemented. |
| let    | =(let [x 1 y 2] (+ x y))=         | No strcmp dispatch — not yet implemented. |
| let*   | =(let* [x 1 y (+ x 1)] (+ x y))=  | No strcmp dispatch — not yet implemented. |
| letrec | =(letrec [f (lambda ...)] (f 0))= | No strcmp dispatch — not yet implemented. |

These are =register_builtins=-only symbols ([[file:../codegen.c::12960]]).

* Null-Coalescing

=(?? expr default)= ([[file:../codegen.c::5763]]) returns =expr= if not =nil=,
=default= otherwise. Scalar types skip the nil check.

(?? (lookup key dict) "default")
(?? (safe-div a b) 0)
#+END_SRC
