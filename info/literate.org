Monad has basic support for literate programming in org-mode files.

* Literate =monadl= files

Files with =.monadl= extension are processed as literate documents.
Code blocks with =#+BEGIN_SRC monad= are extracted and compiled.

* How it works

1. All =#+BEGIN_SRC monad ... #+END_SRC= blocks are extracted
2. Concatenated in document order
3. Compiled as a single Monad program

* Usage

#+BEGIN_SRC sh
monad build doc.monadl -o program
#+END_SRC

* Limitations

- No tangle/untangle — pure extraction
- Only =monad= source blocks are extracted
- Order is document order (no weaving)
