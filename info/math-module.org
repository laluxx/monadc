The =Math= module ([[file:../core/Math.mon]], ~400 lines) provides
mathematical functions and constants, infinite sequences, and
combinatorics. Import with =(import Math)=. See
[[file:../context/core-libs.org][context/core-libs.org]] for the library architecture.

1. Constants

| Constant | Value | Description              |
|----------+-------+--------------------------|
| =pi=     | 3.14159... | Circle constant     |
| =e=      | 2.71828... | Natural log base    |
| =tau=    | 6.28318... | Turn constant (2π)  |
| =phi=    | 1.61803... | Golden ratio        |

#+BEGIN_SRC monad
(import Math)
(show pi)                         ;; 3.14159...
(show (* 2 pi))                   ;; 6.28318... (= tau)
#+END_SRC

#+BEGIN_SRC wisp
import Math
show pi                         ;; 3.14159...
show (* 2 pi)                   ;; 6.28318... (= tau)
#+END_SRC

Use constants from the Math module in arithmetic ([[file:arithmetic.org]]) and
with numeric types ([[file:numeric.org]]).

2. Functions

| Function      | Example                  | Description                |
|---------------+--------------------------+----------------------------|
| =gcd=         | =(gcd 12 8)=             | Greatest common divisor    |
| =lcm=         | =(lcm 6 8)=              | Least common multiple      |
| =factorial=   | =(factorial 5)=          | 5! = 120                   |
| =fib=         | =(fib 10)=               | 10th Fibonacci number      |
| =prime?=      | =(prime? 17)=            | Primality test             |
| =clamp=       | =(clamp 0 100 150)=      | Clamp to [0, 100] → 100   |
| =sin=         | =(sin pi)=               | Sine                       |
| =cos=         | =(cos 0)=                | Cosine                     |
| =sqrt=        | =(sqrt 144)=             | Square root                |
| =pow=         | =(pow 2 10)=             | Power (2^10 = 1024)        |
| =floor=       | =(floor 3.14)=           | Round down → 3             |
| =ceiling=     | =(ceiling 3.14)=         | Round up → 4               |
| =round=       | =(round 3.5)=            | Round to nearest → 4       |
| =abs=         | =(abs -5)=               | Absolute value → 5         |
| =even?=       | =(even? 42)=             | Evenness test              |
| =odd?=        | =(odd? 7)=               | Oddness test               |

#+BEGIN_SRC monad
(import Math)
(show (gcd 12 8))           ;; 4
(show (factorial 5))        ;; 120
(show (clamp 0 100 150))    ;; 100
(show (prime? 17))          ;; True
(show (sqrt 144))           ;; 12.0
#+END_SRC

#+BEGIN_SRC wisp
import Math
show (gcd 12 8)           ;; 4
show (factorial 5)        ;; 120
show (clamp 0 100 150)    ;; 100
show (prime? 17)          ;; True
show (sqrt 144)           ;; 12.0
#+END_SRC

Many math functions wrap C FFI calls ([[file:ffi.org]]) to math.h — =(include "math.h")=
is handled automatically by the module.

3. Infinite Sequences

| Sequence      | Description              |
|---------------+--------------------------|
| =naturals=    | 0, 1, 2, 3, ...          |
| =positives=   | 1, 2, 3, 4, ...          |
| =evens=       | 0, 2, 4, 6, ...          |
| =odds=        | 1, 3, 5, 7, ...          |

These are lazy infinite lists. Use =take= from the Coll module ([[file:coll-module.org]])
to consume:

#+BEGIN_SRC monad
(import Math)
(import Coll)
(show (take 5 evens))        ;; (0 2 4 6 8)
(show (take 3 primes))       ;; (2 3 5)
#+END_SRC

#+BEGIN_SRC wisp
import Math
import Coll
show (take 5 evens)        ;; (0 2 4 6 8)
show (take 3 primes)       ;; (2 3 5)
#+END_SRC

See [[file:collections.org][Collections]] and [[file:list.org][Lists]] for collection operations
that work with infinite sequences.

4. Usage Patterns

**4.1 Combinatorics**

#+BEGIN_SRC monad
(import Math)
(show (factorial 10))        ;; 3628800
(show (gcd 54 24))           ;; 6
(show (lcm 6 8))             ;; 24
#+END_SRC

#+BEGIN_SRC wisp
import Math
show (factorial 10)        ;; 3628800
show (gcd 54 24)           ;; 6
show (lcm 6 8)             ;; 24
#+END_SRC

**4.2 Trigonometry**

#+BEGIN_SRC monad
(import Math)
(show (sin (/ pi 2)))        ;; 1.0
(show (cos pi))              ;; -1.0
#+END_SRC

#+BEGIN_SRC wisp
import Math
show (sin (/ pi 2))        ;; 1.0
show (cos pi)               ;; -1.0
#+END_SRC

**4.3 Rounding**

#+BEGIN_SRC monad
(import Math)
(show (floor 3.99))          ;; 3
(show (ceiling 3.01))        ;; 4
(show (round 3.5))           ;; 4
(show (abs -42))             ;; 42
#+END_SRC

#+BEGIN_SRC wisp
import Math
show (floor 3.99)          ;; 3
show (ceiling 3.01)        ;; 4
show (round 3.5)           ;; 4
show (abs -42)             ;; 42
#+END_SRC

5. Implementation Notes

- Math functions are defined in =core/Math.mon= (~400 lines). Many
  delegates to C math.h functions via the FFI system — =(include "math.h")=
  is handled automatically by the module
  ([[file:../context/core-libs.org][context/core-libs.org — Math module]]).
- Infinite sequences (=naturals=, =evens=, etc.) are lazy lists
  constructed via recursive =cons= of delayed computations.
- Constants (=pi=, =e=, =tau=, =phi=) are computed at compile time
  using high-precision approximations.

6. Key Points

- =(import Math)= provides 15+ functions covering number theory,
  trigonometry, rounding, and combinatorics plus 4 constants.
- Infinite sequences (=naturals=, =evens=, =odds=) work with =take=
  from the Coll module.
- Many functions wrap C math.h calls behind the scenes.
- Known issue: variadic =gcd= fails during compilation.
- Module requires explicit =import Math= — not automatic.

7. Known Issues
