The =Math= module ([[file:../core/Math.mon]], ~400 lines) provides
mathematical functions and constants, infinite sequences, and
combinatorics. Import with =(import Math)=.

* Constants

| Constant | Value | Description              |
|----------+-------+--------------------------|
| =pi=     | 3.14159... | Circle constant     |
| =e=      | 2.71828... | Natural log base    |
| =tau=    | 6.28318... | Turn constant (2π)  |
| =phi=    | 1.61803... | Golden ratio        |

#+BEGIN_SRC monad
(import Math)
(show pi)                         ;; 3.14159..)
(show (* 2 pi))                   ;; 6.28318... (= tau)
#+END_SRC

#+BEGIN_SRC wisp
import Math
show pi                         ;; 3.14159..)
show (* 2 pi)                   ;; 6.28318... (= tau)
#+END_SRC

* Functions

| Function      | Example                  | Description                |
|---------------+--------------------------+----------------------------|
| =gcd=         | =(gcd 12 8)=             | Greatest common divisor    |
| =lcm=         | =(lcm 6 8)=              | Least common multiple      |
| =factorial=   | =(factorial 5)=          | 5! = 120                   |
| =fib=         | =(fib 10)=               | 10th Fibonacci number      |
| =prime?=      | =(prime? 17)=            | Primality test             |
| =clamp=       | =(clamp 0 100 150)=      | Clamp to [0, 100] → 100   |
| =sin=         | =(sin pi)=               | Sine                       |
| =cos=         | =(cos 0)=                | Cosine                     |
| =sqrt=        | =(sqrt 144)=             | Square root                |
| =pow=         | =(pow 2 10)=             | Power (2^10 = 1024)        |
| =floor=       | =(floor 3.14)=           | Round down → 3             |
| =ceiling=     | =(ceiling 3.14)=         | Round up → 4               |
| =round=       | =(round 3.5)=            | Round to nearest → 4       |
| =abs=         | =(abs -5)=               | Absolute value → 5         |
| =even?=       | =(even? 42)=             | Evenness test              |
| =odd?=        | =(odd? 7)=               | Oddness test               |

#+BEGIN_SRC monad
(import Math)
(show (gcd 12 8))           ;;)
(show (factorial 5))        ;; 12)
(show (clamp 0 100 150))    ;; 10)
(show (prime? 17))          ;; Tru)
(show (sqrt 144))           ;; 12.)
#+END_SRC

#+END_SRC

* Infinite Sequences

| Sequence      | Description              |
|---------------+--------------------------|
| =naturals=    | 0, 1, 2, 3, ...          |
| =positives=   | 1, 2, 3, 4, ...          |
| =evens=       | 0, 2, 4, 6, ...          |
| =odds=        | 1, 3, 5, 7, ...          |

These are lazy infinite lists. Use =take= to consume:

#+BEGIN_SRC monad
(import Math)
(import Coll)
(show (take 5 evens))        ;; (0 2 4 6 8)
(show (take 3 primes))       ;; (2 3 5)
#+END_SRC

#+BEGIN_SRC wisp
import Math
import Coll
show (take 5 evens)        ;; (0 2 4 6 8)
show (take 3 primes)       ;; (2 3 5)
#+END_SRC
