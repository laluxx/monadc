| Codegen          | =codegen.c=   | LLVM IR generation           |
| Runtime          | =repl.c=      | JIT execution (SIGSEGV/SIGBUS handlers) |

In the REPL, three recovery mechanisms keep the session alive:

| Mechanism          | Scope                        |
|--------------------+------------------------------|
| =setjmp=/=longjmp= | Parse errors in =REPL_PARSE= macro |
| =g_repl_escape=    | Runtime errors / crashes     |
| =repl_signal_handler= | SIGSEGV / SIGBUS from JIT code |

** Debug Flags

| Flag            | Output                 | Use case               |
|-----------------+------------------------+------------------------|
| =--emit-json=  | AST as JSON            | Inspect parsed form    |
| =--emit-ir=    | LLVM IR text           | Inspect generated code |
| =--emit-asm=   | Assembly text          | Inspect machine code   |

Common error patterns include unbound variables, type mismatches, unbalanced delimiters, and unknown functions. Each error reports =file:line:column= with a source pointer.
