The REPL provides an interactive environment with the following features:

- **Readline**: history navigation (up/down/Ctrl+R), inline editing, persistent history
- **Auto-print**: every expression result is printed automatically after evaluation
- **Multi-line editing**: the parser waits for balanced delimiters before evaluating
- **Input syntax**: both S-expression and Wisp syntax are accepted
- **Error recovery**: setjmp/longjmp escapes catch parse errors; SIGSEGV and SIGBUS signal handlers catch JIT runtime crashes and return to the prompt without exiting
- **SIGINT**: Ctrl+C interrupts a running evaluation

| Feature              | Mechanism                           |
|----------------------+-------------------------------------|
| Parse recovery       | =setjmp= / =longjmp= in =REPL_PARSE= macro |
| Runtime crash        | =repl_signal_handler= for SIGSEGV/SIGBUS  |
| Evaluation abort     | =repl_sigint_handler= for SIGINT          |
| History / editing    | GNU Readline library                      |
