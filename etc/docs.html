Introduction #

Monad is a functional programming language designed for clarity, composition, and type safety. Built around the concept of monads as first-class citizens, it enables developers to write expressive, maintainable code with powerful abstractions.

Philosophy

Monad prioritizes composition over inheritance, immutability over mutation, and explicit effects over hidden side effects.

Key Features

  • Pure Functions - All functions are pure by default, effects are explicit
  • Type Inference - Powerful Hindley-Milner type inference with minimal annotations
  • Pattern Matching - Exhaustive pattern matching with compiler warnings
  • Effect System - Track side effects at the type level
  • Zero-cost Abstractions - High-level concepts compile to efficient machine code

Installation #

Install Monad using the official installer. Supports macOS, Linux, and Windows (WSL).

Terminal
# macOS / Linux
curl -fsSL https://monad.dev/install.sh | sh

# Or using Homebrew
brew install monad-lang

# Verify installation
monad --version

Linux Kernel 5.4+ x86_64, ARM64

Quick Start #

Create your first Monad program. The classic "Hello, World" with a functional twist:

hello.mon
-- Main entry point
module Main where

import IO
import String

-- Pure function greeting
greet : String -> String
greet name = "Hello, " ++ name ++ "!"

-- Main IO action
main : IO ()
main = do
    name <- getLine
    putStrLn (greet name)

Run the program:

Terminal
$ monad run hello.mon
> Monad
Hello, Monad!

Syntax Overview #

Monad uses clean, indentation-sensitive syntax inspired by Haskell and Python, optimized for readability.

Variables and Immutability

variables.mon
-- Immutable bindings (default)
x = 42
name = "Monad"

-- Type annotations (optional)
count : Int
count = 100

-- Mutable references (explicit)
counter : Ref Int
counter = ref 0

Functions

functions.mon
-- Function definition
add : Int -> Int -> Int
add x y = x + y

-- Lambda expressions
square = \x -> x * x

-- Function composition
process = format . validate . parse

-- Higher-order functions
apply : (a -> b) -> a -> b
apply f x = f x

Type System #

Monad features a powerful static type system with full inference. Write code without types, get safety with them.

Basic Types

types.mon
-- Primitive types
intVal : Int        -- 64-bit integer
floatVal : Float      -- 64-bit float
boolVal : Bool       -- true | false
charVal : Char       -- Unicode character
strVal : String     -- UTF-8 text

-- Composite types
list : List Int       -- [1, 2, 3]
pair : (Int, String) -- (42, "answer")
record : { x: Int, y: Int }  -- { x: 1, y: 2 }

Algebraic Data Types

adt.mon
-- Sum types (enums)
data Option a = None | Some a

-- Product types (records)
data Person = Person {
    name : String,
    age : Int
}

-- Recursive types
data Tree a = Leaf | Node a (Tree a) (Tree a)

Monads #

The heart of the language. Monads provide a unified interface for sequencing computations with context.

Core Concept

A monad is a design pattern that allows chaining operations while managing effects, state, or context automatically.

Common Monads

monads.mon
-- Maybe monad for null safety
safeDivide : Float -> Float -> Maybe Float
safeDivide x 0 = Nothing
safeDivide x y = Just (x / y)

-- Result monad for error handling
parseInt : String -> Result ParseError Int

-- IO monad for effects
readFile : Path -> IO (Result IOError String)

-- Do notation for sequencing
program : IO ()
program = do
    putStr "Enter number: "
    input <- getLine
    case parseInt input of
        Ok n -> print (n * 2)
        Err e -> putStrLn "Invalid input"

Operators #

Monad provides a rich set of operators with customizable precedence.

operators.mon
-- Arithmetic
+  -  *  /  %           -- Basic math
**                      -- Exponentiation

-- Comparison
==  /=  <  >  <=  >=    -- Equality and ordering

-- Logical
&&  ||  !               -- And, Or, Not

-- Functional
|>                      -- Forward pipe: x |> f = f x
<|                      -- Backward pipe
>>                      -- Monad bind: flatMap
=<<                     -- Reverse bind
.                       -- Composition: (f . g) x = f (g x)
$                       -- Application: f $ x = f x

-- List
++                      -- Concatenation
::                      -- Cons: x :: xs = [x] ++ xs

Error Handling #

Monad eliminates exceptions. All errors are explicit and type-safe using the Result type.

error.mon
data FileError = NotFound Path 
               | PermissionDenied Path
               | IOError String

readConfig : Path -> Result FileError Config
readConfig path = 
    readFile path
        |> map parseConfig
        |> andThen validateConfig

-- Error propagation with context
loadApp : Result AppError App
loadApp = do
    config <- mapErr ConfigError (readConfig "app.conf")
    db <- mapErr DBError (connectDB config.database)
    Ok (App config db)

FFI Advanced #

Foreign Function Interface allows calling C libraries directly with zero overhead.

ffi.mon
-- Import C functions
foreign import "math.h"
    sin : Float -> Float
    cos : Float -> Float
    sqrt : Float -> Float

-- Safe wrappers
safeSqrt : Float -> Maybe Float
safeSqrt x = 
    if x < 0 
    then Nothing 
    else Just (sqrt x)

Optimization Advanced #

Monad compiles through LLVM with aggressive optimizations enabled by default.

Compiler Flags

terminal
monad build --release          # Optimized build
monad build --target=wasm32    # WebAssembly output
monad build --emit-llvm        # Output LLVM IR
Performance Tips

Use strictness annotations (!) for hot loops. Enable tail-call optimization with -O3. Profile with monad prof.

Deployment #

Compile to native binaries, WebAssembly, or run in the cloud.

deployment.mon
-- Cloud function deployment
module API where

import HTTP
import JSON

-- Auto-generates OpenAPI spec
handler : Request -> IO Response
handler req = do
    body <- jsonBody req
    case validate body of
        Ok user -> jsonResponse 200 (createUser user)
        Err e -> jsonResponse 400 (errorObj e)
