[DEC id:dec.tests.heap-literals.contract date:2026-06-12 by:codex]
  The regression is not only a parser check. The runner must exercise JSON
  emission, compilation, generated executable creation, and execution so future
  changes cannot silently drop =array.is_heap=, print heap arrays as =[]=, or
  break Wisp =for i -> heap= desugaring.

[DEC id:dec.tests.path-literals-split date:2026-06-12 by:codex]
  Path literal compile-time rejection lives in
  =tests/codegen/rt_path_literal_value.mon= and expects a definable =Path= value
  literal= diagnostic for valid Unix path syntax used as a value. Heap array
  runtime coverage stays here so the two behaviors can change independently.
